¿Qué es GitHub Local?

Aqui tienes un Manual de usuario para introductorio a git local .... buena suerte!!.

Definición

GitHub Local se refiere al uso de Git en tu máquina local para controlar versiones y luego sincronizar los cambios con tu repositorio remoto en GitHub.

Pasos básicos

  1. Instala Git en tu computadora.
  2. Configura Git con tu nombre y correo:
    3. git config --global user.name "Tu Nombre"
    git config --global user.email "tucorreo@ejemplo.com"
  3. Clona un repositorio o inicia uno nuevo:
    4. git clone https://github.com/usuario/repositorio.git
    git init
  4. Agrega y confirma tus cambios:
    5. git add .
    git commit -m "Primer commit"
  5. Conecta tu repositorio remoto (si no está):
    6. git remote add origin https://github.com/usuario/repositorio.git
  6. Sube tus cambios a GitHub:
    7. git push -u origin main
